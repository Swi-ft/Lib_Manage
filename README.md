# Library Management System

## Overview
The Library Management System is a console-based application designed to manage book lending operations for students and faculty. It includes features for borrowing, returning books, and tracking overdue fines.

## Features
- Borrow and return books
- Fine calculation for overdue books
- File I/O for data persistence
- Support for students and faculty accounts
- Librarian access for administrative tasks

## Installation
1. Clone the repository:
   ```sh
   git clone git clone https://github.com/yourusername/Lib_Manage.git
   cd library-management-system
   ```

2. Run the executable:
   ```sh
   ./main.exe  # On Windows
   ./main      # On Linux/macOS
   ```

## Usage
- Run the application and follow the prompts to borrow or return books.
- The system will automatically track fines for overdue books.
- All data is stored persistently using file I/O.
